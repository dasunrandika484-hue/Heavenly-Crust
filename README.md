# 🍰 Devinda's Bakery App 🚀

Welcome to the **Devinda's Bakery App**! This is a modern, mobile-friendly application designed for a bakery to showcase its delicious treats and manage customer orders seamlessly.

---

## 📱 Project Overview
This app was built using **Glide (No-Code)** technology. It allows customers to browse through various bakery items, view prices, and place orders directly through an intuitive interface.

### ✨ Key Features:
* **Visual Catalog:** Beautifully displayed items like Chocolate Cake, Muffins, and Burgers with high-quality images.
* **Detailed Descriptions:** Each item comes with a short description to entice customers.
* **Order Management:** Integrated order placement system (Orders table).
* **Mobile-First Design:** Fully responsive and works like a native app on any smartphone.

---

## 🛠️ Technology Stack
* **Platform:** [Glide](https://www.glideapps.com/)
* **Database:** Glide Tables (Internal)
* **Assets:** Cloud-hosted images for fast loading.
